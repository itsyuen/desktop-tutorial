# Python Source directory in active workspaces

[![hackmd-github-sync-badge](https://hackmd.io/YTA1QD7HT3agcgWDzs6RMw/badge)](https://hackmd.io/YTA1QD7HT3agcgWDzs6RMw)

This README describes the Python directory. Currently a practice ground for Python code to be archived for code snippets of Python that translates algorithm/control strategy/hardware-firmware-software management glue that is used in ROS2 system that make and maintain how memory flow and node/message/topic flow is controlled.

This directory is intended to become a working area for Python code snippets that will later be incorporated into ROS2, STM32, Arduino, and other hardware-related intermediate files. This will be later a most important working ground alongside `bash` commands that will later be used to call out all the needed ROS2 commands, and may be heavy.

Currently my `python` will mostly be in `learning` repo, and thus this directory might grow with ROS2 practice. Stand-alone Python tutorials will be in `learning`.

## Possible Directory Architecture

Later will add Python subdirectory of:

- `scripts/`: Utility scripts for automation and setup.
- `nodes/`: ROS2 node implementations in Python.
- `packages/`: Colcon-compatible ROS2 packages with Python modules.
- `examples/`: Sample code snippets for algorithms and control strategies.
- `tests/`: Unit tests and integration tests for Python code.
- `docs/`: Documentation and tutorials specific to Python in ROS2.
